Fizz Buzz is a common and classic coding excersize in which the program should loop through the numbers 1-100 and print out:
fizz, if the number is divisible by 3
buzz, if the number is divisible by 5
fizzbuzz, if the number is divisible by 3 and 5 

            for (i = 1; i <= 100; i++) {
                if (i % 3 === 0 && i % 5 === 0) {
                    FizzBuzz
                } else if (i % 3 === 0) {
                    Fizz
                } else if (i % 5 === 0) {
                    Buzz
                }
            }

            A simple loop through 100 iterations will allow us to check each number
            We start with numbers divisible by both 3 and 5 for correct behavior 
            when using a 'for' loop for this excersize.
